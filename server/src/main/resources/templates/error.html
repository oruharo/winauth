× 認証エラー

Windows統合認証に失敗しました。

解決方法

以下の点を確認してください：

  • ドメインアカウントでWindowsにログインしているか
  • ブラウザがWindows統合認証をサポートしているか
  • ブラウザのセキュリティ設定で、このサイトが信頼されたサイトに追加されているか
  • 企業ネットワーク内からアクセスしているか

Internet Explorer / Microsoft Edgeの場合

  1. 「インターネットオプション」を開く
  2. 「セキュリティ」タブ → 「ローカルイントラネット」を選択
  3. 「サイト」をクリックし、このサイトのURLを追加
  4. 「詳細設定」で「イントラネットのサーバーに対してのみ自動的にログオンする」を有効化
